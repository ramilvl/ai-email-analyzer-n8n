# AI Email Analyzer (n8n)

An AI-powered email analysis workflow built with self-hosted n8n, Docker, OpenRouter, Google Sheets, and Telegram.

## Features

- Monitors incoming Gmail messages
- Analyzes emails using an LLM (OpenRouter)
- Classifies email category
- Assigns priority level
- Generates a summary
- Stores results in Google Sheets
- Sends Telegram notifications

## Workflow

Gmail Trigger
→ OpenRouter AI
→ JSON Parsing
→ Google Sheets
→ Telegram

## Tech Stack

- n8n
- Docker
- OpenRouter
- Gmail API
- Google Sheets API
- Telegram Bot API
